# 个人静态博客 🌿
基于 **Hugo** 搭建的轻量化个人博客，部署在 GitHub Pages，简洁清爽、访问流畅。

## 🌐 在线访问
博客地址：https://nightfire168.github.io

## ✨ 网站功能
- 首页展示个人介绍、精选文章、风景美图
- 首页默认只展示 3 篇最新文章，点击「显示更多」查看全部文章
- 美景板块默认展示 3 张图片，点击「显示更多」浏览全部摄影作品
- 文章弹窗阅读、图片放大预览
- 响应式布局，电脑/手机均可正常访问

## 🛠️ 技术栈
- 搭建框架：Hugo
- 前端样式：原生 CSS / HTML / JavaScript
- 部署平台：GitHub Pages
- 内容管理：Markdown 编写文章

## 📁 目录说明
my-blog/
├── content/posts/ # 所有文章存放目录（Markdown）
├── static/ # 静态资源、本地图片
├── themes/ # 博客主题样式文件
├── hugo.toml # 网站配置、美图列表、基础设置
└── README.md # 项目说明文件


## 📝 如何新增内容
### 1. 添加文章
1. 进入 `content/posts/` 文件夹
2. 新建 `.md` 格式文件，复制文章模板编写内容
3. 填写标题、描述、封面图、唯一 pid
4. 保存后执行同步命令，自动更新至线上

### 2. 添加美景图片
1. 打开根目录 `hugo.toml`
2. 找到 `gallery` 数组，新增图片链接/本地图片路径
3. 保存文件，同步即可
